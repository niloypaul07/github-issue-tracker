1️⃣ What is the difference between var, let, and const?

Answer- var is function-scoped and hoisted, let is block-scoped and mutable, and const is block-scoped and immutable.

2️⃣ What is the spread operator (...)?

Answer- The spread operator  is a way to copy or separate items from an array or object easily.

3️⃣ What is the difference between map(), filter(), and forEach()?

Answer- map() makes a new array by changing each item, filter() makes a new array with only items that meet a condition, and forEach() just does something with each item without making a new array.

4️⃣ What is an arrow function?

Answer- An arrow function is a shorter way to write a function using => instead of function.

5️⃣ What are template literals?

Answer- Template literals are special strings that let you add variables and create multi-line text using backticks `.
